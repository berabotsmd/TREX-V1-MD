## TREX-V1-MD 

#### Run Locally

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/berabotsmd/TREX-V1-MD.git
   cd TREX-V1-MD 
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Configuration**:

   ```bash
   echo 'SESSION_ID="your session id here"' > .env
   ```

4. **Start the Bot**:

   ```bash
   npm start
   ```
